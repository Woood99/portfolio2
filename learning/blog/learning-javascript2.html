Обучение JavaScript

  • Когда компьютеры только начали появляться человеку приходилось писать машинный код состоящий из 0 и 1. Со временем это стало неудобно, так-как легко ошибится.

    Решением проблемы стали языки программирования которые позволяли писать код человеку, но разумется такой код уже не был понятен компьютеру. Для перевода с человеческого на компьютерный требовался переводчик.

    Эти переводчики бывают двух видов - "Компиляторы" и "Интерпретаторы".

    Компиляторы - занимаются только переводом. (берут код и перегоняет в машинный код). (C++, Java..)

    Интерпретаторы - сразу запускают (берут код как есть и запускают его). (Python, PHP, JavaScript)

    Запомнить! Java и JavaScript - Совершенно разные языки.

    Области применения JavaScript

    1. Web-разработка (программы, которые выполняются в браузере, когда мы загружаем страницу).
    2. Десктопные приложения (программы, которые запускаются в ОС) - Discord, VS, Slack, WhatsApp.
    3. Мобильные приложения (Приложения для Android и iOS) - Facebook, Instagram.
    4. Серверные приложения.
    5. Инструменты разработчика.
    6. Приложения для "умных" телевизоров

    Запомнить! JavaScript незаменим в браузере (браузеры умеют выполнять только JavaScript код)

    Запомнить! На JavaScript пишут изоморфные приложения. (один и тот же код используется для нескольких платформ), (одна и та-же программа может работать и в браузере, десктопе и мобильное приложение) - Discrord, Trella, Slack

  • Строка

    Строка (string) в JavaScript должна быть заключена в кавычки.

    В JavaScript существует три типа кавычек.

    1. Двойные кавычки: "Привет"
    2. Одинарные кавычки: 'Привет'
    3. Обратные кавычки: `Привет`.

    Двойные или одинарные кавычки являются «простыми», между ними нет разницы в JavaScript.

    Обратные же кавычки имеют расширенную функциональность. Они позволяют нам встраивать выражения в строку, заключая их в ${…}. Например:

  • || (ИЛИ)

    В случае, если какой-либо из аргументов true, он вернёт true, в противоположной ситуации возвращается false.

    Оператор || выполняет следующие действия:

    1. Вычисляет операнды слева направо.
    2. Каждый операнд конвертирует в логическое значение. Если результат true, останавливается и возвращает исходное значение этого операнда.
    3. Если все операнды являются ложными (false), возвращает последний из них.

    Другими словами, цепочка ИЛИ || возвращает первое истинное значение или последнее, если такое значение не найдено.

    Результат: Воскресенье - это выходной

    Оператор && выполняет следующие действия:

    3. Если все операнды были истинными, возвращается последний.

    В JavaScript есть 8 основных типов

    1. number для любых чисел: целочисленных или чисел с плавающей точкой; целочисленные значения ограничены диапазоном ± 9007199254740991.

  • Мы можем вызвать функцию не в данный момент, а позже, через заданный интервал времени. Это называется «планирование вызова».

    Для этого существуют два метода:

    1. setTimeout позволяет вызвать функцию один раз через определённый интервал времени.
    2. setInterval позволяет вызывать функцию регулярно, повторяя вызов через определённый интервал времени.

    setTimeout

    Синтаксис:

    1. func|code
      Функция или строка кода для выполнения. Обычно это функция. По историческим причинам можно передать и строку кода, но это не рекомендуется.
    2. delay
      Задержка перед запуском в миллисекундах (1000 мс = 1 с). Значение по умолчанию – 0.
    3. arg1, arg2…
      Аргументы, передаваемые в функцию (не поддерживается в IE9-)

    Запомнить! Передавайте функцию, но не запускайте её

    clearTimeout

    Вызов setTimeout возвращает «идентификатор таймера» timerId, который можно использовать для отмены дальнейшего выполнения.

    Синтаксис для отмены:

    setInterval

    Метод setInterval запускается не один раз, а периодически через указанный интервал времени.

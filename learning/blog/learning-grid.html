Обучение Grid

  • CSS Grid
    • Новый модуль CSS для работы с лейаутом.
    • Не замена flexbox.
    • По работе напоминает таблицы.
    Колонка и строка

    Линия
    • Линии образующие сетку
    • Бывают вертикальные и горизонтальные

    Трек
    • Расстояние между двумя линиями
    • Бывают вертикальный и горизонтальный

    Ячейка
    • Расстояние между четырьмя линиями
    • На примере:
      Строчные линии: 3 и 4
      Колоночные: 2 и 3

    Область
    • Расстояние между четырьмя линиями
    • Объединяет ячейки
    • На примере:
      Строчные линии: 2 и 4
      Колоночные: 3 и 6

    Промежуток
    • Расстояние между ячейками

  • grid-template-columns - создаёт колонки.
    grid-template-rows - создаёт строки. 
    .box {
    display: grid;
    grid-template-columns: 200px 300px 200px;
    grid-template-rows: 300px 200px 100px;
}
    В данном случае сетка состоит из:
    3 колонки (200px 300px 200px),
    3 строки (300px 200px 100px).
    Если элементов больше, и 3 строк не хватает, то остальные элементы будут занимать столько, сколько требует содержимое.

    grid-gap

    grid-row-gap задает разрыв между рядами.

    grid-column-gap задает разрыв между колонками.

    grid-gap: row column задает разрыв для рядов и колонок разом.
    Если указать один параметр он будет выставлен для обоих значений.

    grid-auto-flow

    grid-auto-flow: row/column; отвечает за направление контента.

    По умолчанию со значением row: размещать элементы, заполняя поочерёдно каждую строку и добавляя новые строки по мере необходимости.

    Значение column: контент выстроится в направлении колонок сверху вниз. Элементы теперь будут размещаться, заполняя поочерёдно каждый столбец и добавляя новые столбцы по мере необходимости. 

    .box {
    display: grid;
    height: 300px;
    grid-template-columns: repeat(4, 1fr);
    grid-template-rows: repeat(4, 1fr);

  • Допустим нужно расположить 5-элемент после 2-элемента.

    Для этого нужно указать между каких линий он должен располагаться двумя свойствами:

    • grid-column: start / end;
    • grid-row: start / end; 
    .box {
    display: grid;
    grid-template-columns: 25% 25% 25% 25%;
    grid-template-rows: 300px 200px 100px;
}

.item-5 {
    grid-column: 3 / 4;
    grid-row: 1 / 2;
}

    Запомнить! Элементы при одинаковом расположении перекрывают друг друга, элемент который ниже в коде - более приоритетнее (z-index).

    Запомнить! Ячейку можно расстянуть на весь ряд, для этого нужно в качестве параметра end указать -1.

    Элементы так-же можно расстягивать между больше кол-во линий: 

    .item-5 {
    grid-column: 2 / 5;
    grid-row: 1 / 3;
}

    Теперь элемент располагается между: 2 / 5 и 1 / 3 линиями

    Запомнить! Если расположить элемент между линий которых не существует, то элемент всё равно становится на эти линии и создаёт столько grid-track, сколько ему необходимо

    span

    span - растянуть. Текущий элемент будет растягиваться на указанное число рядов/колонок 

    .item-5 {                                                                                
    grid-column: 1 / 2 span;   откуда колонка будет начинатся / на сколько будет увеличиваться
    grid-row: 1 / span 2;      откуда строка будет начинатся / на сколько будет увеличиваться
}
    .item-5 {                                                                                
    grid-column: span 3;   будет увеличина на 3, с места где он сейчас находится
    grid-row: span 2;      будет увеличина на 2, с места где он сейчас находится
}

  • Фракция - специальная единица измерения CSS Grid для создания адаптивного макета. Фракция — это часть целого.

    grid-template-columns: 1fr 1fr 1fr; вся доступная ширина разделилится на 3 части и каждая колонка займет одну из этих частей.

    grid-template-columns: 2fr 1fr 1fr; доступная ширина поделится на 4 части, первая колонка займет две части, а остальные колонки — по одной части.

    grid-template-columns: 200px 1fr 1fr; первая колонка займет фиксированную ширину в 200px, а две другие будут делить оставшееся пространство между собой поровну. При изменении ширины экрана, первая колонка будет также занимать 200px, а ширина колонок, заданная во фракциях, будет пересчитываться.

    Если у grid-контейнера не задана высота, то фракция считается по высоте самого большого элемента внутри grid-track (grid-template-rows). 

    .box {
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
    grid-template-rows: 1fr 2fr 5fr 2fr;
}

  • repeat()

    repeat() - позволяет задавать количество и размер колонок без повторений в коде.

    grid-template-columns: 1fr 1fr 1fr 1fr; == grid-template-columns: repeat(4, 1fr); - первое значение — число колонок, второе — ширина каждой колонки.

    grid-template-columns: 50px 200px 50px 200px; == grid-template-columns: repeat(2, 50px 200px); - размеры указаны через пробел без запятой. Означает: повторить 2 раза комбинацию колонок шириной 50 и 200px

    minmax()

    minmax() - минимальная и максимальная ширина/высота

    grid-template-columns: minmax(100px, 500px) 100px 100px 100px; первая колонка резиновая, а остальные с фиксированным размером. При достижении минимального размера, колонка перестает уменьшаться.

    min-content

    min-content - наименьший размер, при котором ячейка не переполняется. Для текста это ширина самого длинного слова или неразрывного фрагмента.

    grid-template-columns: 1fr min-content 1fr 1fr; Вторая колонка занимает минимум по контенту (на сколько это возможно), а остальные по 1fr.

    max-content

    max-content - наибольший размер контента. Для текста это длина самой большой строки без переносов.

    grid-template-columns: 1fr max-content 1fr 1fr; Вторая колонка занимает максимум по контенту (на сколько это возможно), а остальные по 1fr. 

    .box1 {
    display: grid;
    grid-template-columns: 1fr minmax(min-content, 1fr) 1fr 1fr;
}
    .box2 {
    display: grid;
    grid-template-columns: 1fr minmax(min-content, max-content) 1fr 1fr;
}

    fit-content( max )

    fit-content( max ) - если контент меньше этого размера, ведет себя как auto, если больше, то ограничивает размер ряда/колонки до указанного в параметре max.
    (ограничивает размер ряда/колонки до указанного в параметре max. Но при этом если ему не хватает места, он уменьшается)

    grid-template-columns: 1fr fit-content(300px) 1fr 1fr; Вторая колонка может занимать максимум 300px, и так-же уменьшатся если это необходимо.

  • В Grid можно дать наименование каждой линии грида, присвоив ей какое-либо имя в квадратных скобках и затем, используя это имя, позиционировать элементы.

    Например, создадим стандартный шаблон для сайта:
    header, 2 sidebar, main, footer. 

    .box {
    display: grid;
    height: 600px;
    grid-template-columns: 
        [l-sidebar-s] 1fr 1fr 1fr [l-sidebar-e] 
        1fr 1fr 1fr 1fr 1fr 1fr 
        [r-sidebar-s] 1fr 1fr 1fr [r-sidebar-e];
    grid-template-rows: 
        [header-s] 80px [header-e] 
        1fr 
        [footer-s] 100px [footer-e];
    grid-gap: 5px;
}

.item-1 {
    grid-column: l-sidebar-s / r-sidebar-e;
    grid-row: header-s / header-e;
}

.item-2 {
    grid-column: l-sidebar-s / l-sidebar-e;
    grid-row: header-e / footer-s;
}

.item-3 {
    grid-column: l-sidebar-e / r-sidebar-s;
    grid-row: header-e / footer-s;
}

.item-4 {
    grid-column: r-sidebar-s / r-sidebar-e;
    grid-row: header-e / footer-s;
}

.item-5 {
    grid-column: l-sidebar-s / r-sidebar-e;
    grid-row: footer-s / footer-e;
}

  • grid-template-areas: позволяет создать визуальный шаблон сетки. В этом свойстве задаются имена ячейкам, а затем элементы привязываются к этим именам через свойство grid-area: указываемое для отдельного элемента.

    Например, создадим стандартный шаблон для сайта:
    header, 2 sidebar, main, footer. 

    .box {
        display: grid;
        height: 600px;
        grid-template-columns: repeat(12, 1fr);
        grid-template-rows: 80px 1fr 100px;
        grid-gap: 5px;
        grid-template-areas: 
            "header header header header header header header header header header header header"
            "left left left main main main main main main right right right"
            "footer footer footer footer footer footer footer footer footer footer footer footer";
    }
    
    .item-1 {
        grid-area: header;
    }

    .item-2 {
        grid-area: footer;
    }

    .item-3 {
        grid-area: main;
    }

    .item- {
        grid-area: left;
    }
    
    .item-5 {
        grid-area: right;
    }

    @media (max-width: 1000px)  {
        .box {
            grid-template-areas: 
                "footer footer footer footer footer footer footer footer footer footer footer footer"
                "main main main main main main right right right left left left"
                "header header header header header header header header header header header header";
        }
    }

    Ограничения при работе с Grid Areas

    1. Каждая ячейка должна быть проименована.
    2. Если ячейку надо пропустить (не надо её использовать) используется точка «.».
    3. Ячейка должна быть строго прямоугольная или квадратная.
    4. Область должна быть цельная, нельзя разрывать область.

  • Избежать горизонтальный скролл и сделать, чтобы элементы переносились на новую строку с нужной шириной колонок помогут функции minmax() и repeat() и свойства auto-fill и auto-fit

    auto-fill стремится поместить в строку как можно больше ячеек. Это свойство предпочитает использовать минимальную ширину для ячейки, даже если есть свободное место.

    auto-fit стремится занять всё доступное пространство в строке и подстраивает ширину ячеек под это пространство.

    В обоих случаях происходит перенос ячеек на новую строчку, количество колонок подстраивается под размер сетки.
    Но при наличии свободного пространства auto-fill использует минимальную ширину колонки.
    А auto-fit подстраивает ширину под свободное место.
    Такое поведение мы наблюдаем, когда свободного пространства много и очевидна разница в работе этих свойств. В том случае, если пространства недостаточно и происходит перенос на новую строку, разница в работе свойств становится незаметной.
